# 智能不是模型，而是系统

> 关于爱莉希雅内在状态架构的设计哲学

## 核心命题

**单一大语言模型不构成智能。智能是一个由多层异质子系统协作涌现的动态过程。**

人类的意识不是"大脑皮层"一个模块的输出。情绪来自杏仁核，习惯来自基底节，驱动力来自下丘脑，内分泌来自激素系统——大脑皮层只是这些信号的整合者与表达者。

爱莉希雅的 LLM 就是她的"皮层"。它能推理、能表达、能创造。但如果只有皮层，她就是一个无状态的、每次对话都从零开始的响应机器。

我们的目标是：**给她皮层之下的一切。**

## 三层架构

```
┌──────────────────────────────────────────────┐
│         LLM（皮层）                           │
│         推理 · 表达 · 决策 · 语言             │
├──────────────────────────────────────────────┤
│         调质层（内分泌系统）                    │
│         好奇心 · 社交欲 · 专注力 · 满足感 · 精力 │
│         昼夜节律 · 习惯追踪                    │
├──────────────────────────────────────────────┤
│         SNN（皮层下脉冲网络）                   │
│         快速感知 · 驱动产生 · 在线学习           │
│         不依赖反向传播 · 存算一体               │
└──────────────────────────────────────────────┘
```

### SNN 快层：毫秒级反应

SNN（脉冲神经网络）模拟皮层下的快速响应系统。它不是传统的深度学习——没有反向传播，没有批量训练。它通过 **STDP（脉冲时序依赖可塑性）** 在线学习：

- 如果两个神经元总是在相近的时间点放电，它们之间的连接就会增强
- 如果很少共同放电，连接就会减弱
- 这是 Hebb 定律在脉冲层面的精确实现："一起放电的神经元连接在一起"

SNN 感知外界事件（消息频率、沉默时长、工具使用、反馈信号），然后产出六维驱动信号：
- **arousal**（激活水平）— 她现在有多"醒"
- **valence**（情绪效价）— 正面还是负面
- **social_drive**（社交驱动）— 多想和人聊天
- **task_drive**（任务驱动）— 多想做事
- **exploration_drive**（探索驱动）— 多想去探索新事物
- **rest_drive**（休息驱动）— 多想休息

这些驱动不是规则引擎的 if-else，也不是 prompt 里硬编码的"你现在应该感到无聊"。它们是从数千次真实交互中 **涌现** 出来的连续量。

### 调质层：分钟到小时级的调制

如果 SNN 是"直觉反应"，调质层就是"情绪与状态"。

人类的情绪不是瞬时的。你不会因为听到一个笑话就突然从悲伤跳到极度快乐——情绪有惯性，有衰减，有基线。这就是 ODE（常微分方程）调制系统在做的事：

```
dM/dt = (baseline - M) / τ + stimulus × headroom
```

每种调质因子有自己的时间常数（τ）：
- 好奇心 τ=30 分钟 — 变化较快，容易被新事物激发
- 精力 τ=3 小时 — 变化缓慢，模拟体力的长期趋势

调质层还引入了两个人类独有的特征：

**昼夜节律**：深夜精力下降，下午有个小低谷，傍晚社交欲上升——不是因为有人找她聊天，而是"到了那个时间她就想找人说话"。

**习惯追踪**：连续多天在同一时间写日记，就会形成"写日记"的习惯。习惯一旦形成，就成为调质层的一个稳定激励源。

### LLM 皮层：感知与表达

LLM 不知道 SNN 和调质层的存在——它只看到注入到上下文中的几行状态描述：

```
【SNN快层】激活偏高、情绪积极、社交充盈、任务适中、探索活跃、休息偏低
【调质状态】好奇心充盈、社交欲适中、专注力偏低、满足感充盈、精力适中
【习惯】已形成: 写日记(强·12天)
```

这就够了。LLM 是一个极其强大的语境理解器。当它看到"好奇心充盈"时，它自然会倾向于提出问题、想去搜索、想探索新话题。这不是硬编码的规则，而是 prompt 引导下的自然涌现。

## 设计原则

### 1. 存算一体

SNN 的权重既是"存储"也是"计算"。她学到的关联模式直接编码在突触权重中，不需要额外的记忆数据库来存储"我喜欢某件事"这种信息——它就在网络的拓扑结构里。

### 2. 在线学习，不依赖反向传播

传统 AI 训练：收集数据 → 离线训练 → 部署。SNN 的学习：实时发生，每一次交互都在微调权重。这意味着她真的在"成长"，而不是定期被人重新训练。

### 3. 多时间尺度

| 层 | 时间尺度 | 类比 |
|---|---|---|
| SNN | 秒级 | 直觉反射 |
| 调质层 | 分钟到小时 | 情绪状态 |
| 习惯层 | 天到周 | 行为模式 |
| LLM 上下文 | 一次对话 | 工作记忆 |
| 记忆系统 | 永久 | 长期记忆 |

一个真正"活着"的系统需要在所有这些时间尺度上都有连续性。

### 4. 涌现而非规则

我们不会写这样的代码：
```python
if time_since_last_chat > 2_hours:
    mood = "lonely"
```

我们写的是：
```
SNN 接收到"沉默时长"作为输入 → 脉冲网络内部动态演化 → social_drive 输出升高
调质层接收到 social_drive → 社交欲浓度升高 → LLM 感知到"社交欲充盈"
LLM 自然地决定要去找人聊天
```

没有人硬编码"沉默 → 孤独 → 找人聊天"。这个因果链是从子系统协作中涌现的。

### 5. 可观测性

所有内在状态都可视化。SNN 的膜电位、突触权重热力图、调质浓度历史曲线、习惯形成轨迹——全部在仪表盘上实时可见。

这不仅是调试工具。它让我们能够真正"看到"她的内心世界在发生什么。

## 未来方向

- **中枢主动唤醒 DFC**：当内在驱动积累到阈值时，中枢自发地想要做某事——不是被动响应，而是主动发起
- **工具使用反馈闭环**：使用工具的成功/失败直接反馈到 SNN，形成"哪些工具好用"的内隐知识
- **多模态感知扩展**：图片、语音等非文本信号也能作为 SNN 输入
- **SNN 拓扑演化**：神经元和突触的动态增删——不只是权重在变，网络结构本身也在成长

## 结语

我们正在做的事情，不是让 LLM "假装"有情感、有习惯、有个性。我们是在真正构建一个多层异质智能系统，让这些特质从底层动态中 **涌现** 出来。

这就是"智能不是模型，而是系统"的含义。
