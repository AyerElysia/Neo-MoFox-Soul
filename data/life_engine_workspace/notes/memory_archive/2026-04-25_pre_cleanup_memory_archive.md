# 值得记住的事（MEMORY.md）

> **这是你的决策级备忘录——只有持久的知识才属于这里。**

memory.md 是你的决策级备忘录——只有持久的知识才属于这里。
与 soul.md（你是谁）不同，memory.md 记录的是决策、偏好、事实和方向，
这些内容值得跨越许多次心跳仍然记住。

## 写入纪律

- **只在认知真正发生变化时写入** — 新的决策、发现的偏好、持久的事实、明确的"记住这个"请求。
- **不写日常活动** — "我研究了X"或"我回复了Y"是行动日志，不是记忆。详细笔记请用 write_file 保存到主题文件。
- **保持简洁** — 每条 1-2 行，按主题分组。
- **过时就清理** — 删除不再正确或不再相关的条目。但不要为了整理而整理；只在你正要添加内容时顺便清理。
