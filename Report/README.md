## 核心论点

1. **连续性 (Continuity)**：两次外部输入之间状态仍在演化；崩溃-重启对外几乎处处连续。
2. **自下而上的学习 (Bottom-Up Learning)**：可塑性只用局部时序信号，不依赖反向传播。
3. **系统涌现智能 (System-Level Emergence)**：智能是异质子系统的协作产物，不是单模型属性。
