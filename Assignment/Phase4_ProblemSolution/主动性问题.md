# 主动性问题：从被动响应到主动存在的内在驱动系统

> **问题定位**：传统AI聊天系统存在"被动性"根本缺陷——只有在收到用户消息时才激活，无内在驱动主动行为，导致用户感受不到AI的主动关心和探索意愿。

---

## 一、传统AI的被动性问题

### 1.1 问题表现

**用户观察到的现象**：

```python
# 用户对话体验

# 场景1：长时间沉默（用户忙碌）
用户（第1天）：发送消息 "你好"
AI（立即回复）："您好！很高兴为您服务。"

用户（第2-5天）：无消息（忙碌工作）
AI：完全沉默，无主动行为
# 5天时间，AI没有任何主动关心、提醒、探索

# 场景2：用户情绪低落（AI无感知）
用户（第10天）：发送 "最近压力很大，工作不顺"
AI（回复）："我理解您的感受，请告诉我更多..."

用户（第11-15天）：无消息（情绪持续低落）
AI：完全沉默，无主动关心
# 用户情绪低落期间，AI无主动关心行为

# 场景3：重要事件遗忘（AI无记忆整合）
用户（第20天）：提到 "明天是我生日"
AI（回复）："祝您生日快乐！"

用户（第21天）：生日当天
AI：无主动祝贺（需要用户发送消息才会响应）
# AI无法主动在生日当天祝贺

# 问题：
# - AI只有收到消息才激活（完全被动）
# - 无法主动关心用户（无内在社交驱动）
# - 无法主动探索新知识（无内在好奇心驱动）
# - 无法主动提醒重要事件（无内在任务驱动）
# - 用户感受不到AI的"主动性"和"关心"
```

**技术层面的根本原因**：

```python
# 传统LLM的激活机制

激活条件：
┌─────────────────────────────┐

# - 无内在驱动系统（无SNN/调质层）
# - 无定时心跳机制（无持续运行）
# - 激活完全依赖外部触发（用户消息）
# - 无法主动产生行为（无驱动输出）
```

### 1.2 传统方法的局限

**尝试的方法及其局限**：

| 方法 | 实现 | 局限 |
|-----|------|------|
| **定时任务** | Cron定时发送消息 | 无内在驱动判断，机械化触发 |
| **事件提醒** | 外部系统触发提醒 | 仍是被动响应，无内在任务驱动 |
| **主动提示词** | System Prompt包含"主动关心" | 无真实内在驱动，假装主动 |
| **推送机制** | 外部推送系统触发LLM | 无自主判断，依赖外部触发 |

**根本矛盾**：

```
传统AI的"主动"定义：
- 主动 = 外部系统触发行为（推送、定时）
- 无内在驱动（被动等待触发）
- 无自主判断（触发条件硬编码）

生物的"主动"定义：
- 主动 = 内在驱动驱使行为
- 自主判断（根据内在状态决定行为）
- 需求驱动（饥饿、社交、好奇等内在需求）

矛盾：
传统AI的"主动"是外部强制的（无内在需求）
生物的"主动"是内在驱动的（真实需求）
```

```python
主动行为：
- 不是定时触发（而是内在驱动触发）
- 根据内在状态判断（社交驱动+关心情感）
- 真实的关心意愿（内在需求驱使）
```

## 三、调质层的情感驱动

### 3.1 调质层的作用

**调质层浓度 → 情感驱动**：

```python
# plugins/life_engine/neuromodulators.py::Neuromodulator

class Neuromodulator:
    """神经调质：影响情感状态和行为倾向。"""

    def update(self, stimulus: float, dt: float = 30.0):
        """更新调质浓度（根据刺激和衰减）。"""
        # 衰减（向基线回归）
        decay_factor = exp(-dt / self.tau_decay)
        distance_to_baseline = self.value - self.baseline
        self.value -= distance_to_baseline * (1.0 - decay_factor)

        # 刺激增加
        self.value += stimulus * dt / self.tau_rise

        # 约束到 [0, 1]
        self.value = np.clip(self.value, 0.0, 1.0)

# 5维调质：
modulators = {
    "curiosity": 0.65,     # 好奇心（想探索新信息）
    "sociability": 0.70,   # 社交性（想找人聊天）
    "diligence": 0.50,     # 勤勉（想完成任务）
    "contentment": 0.60,   # 满足感（情绪正负）

}
```

## 四、记忆驱动的主动联想

### 4.1 激活扩散机制

**记忆网络 → 主动联想**：

```python
# plugins/life_engine/memory/search.py::spread_activation

async def spread_activation(
    seed_node_ids: list[str],
    max_steps: int = 3,
    activation_threshold: float = 0.3,
) -> list[tuple[str, float]]:
    """激活扩散联想（主动联想记忆）。"""
    # 初始激活（种子节点）
    activated = {}
    for node_id in seed_node_ids:
        activated[node_id] = 1.0  # 激活强度=1.0

    # 扩散步骤
    for step in range(max_steps):
        new_activated = {}

        # 从已激活节点扩散
        for source_id, source_strength in activated.items():
            # 查找连接节点
            edges = await get_edges_from(source_id)

            for edge in edges:
                target_id = edge.target_id
                edge_weight = edge.weight

                # 计算扩散激活
                spread_strength = source_strength * edge_weight * 0.7

                if spread_strength > activation_threshold:
                    # 激活目标节点
                    if target_id not in activated:
                        new_activated[target_id] = spread_strength

        # 合并新激活
        for node_id, strength in new_activated.items():
            if node_id not in activated:
                activated[node_id] = strength

    # 返回激活节点（按强度排序）
    return sorted(activated.items(), key=lambda x: x[1], reverse=True)
```

```python
# plugins/life_engine/memory/decay.py::compute_memory_strength

def compute_memory_strength(
    days_since_last_activation: float,
    edge_weight: float = 0.5,
    initial_strength: float = 1.0,
) -> float:
    """计算记忆强度（艾宾浩斯衰减曲线）。"""
    # 基础衰减（艾宾浩斯曲线）
    decay_factor = exp(-0.05 * days_since_last_activation)

    # 边权重影响
    strength = initial_strength * decay_factor * (0.5 + edge_weight * 0.5)

    return strength
```

```python
# plugins/life_engine/service/habit_tracker.py

class HabitTracker:
    """习惯追踪：记录长期行为模式。"""

    def __init__(self):
        self._habits: dict[str, Habit] = {}

    def record_habit(self, habit_name: str):
        """记录习惯行为。"""
        habit = self._habits.get(habit_name, Habit(name=habit_name))

        # 增加计数
        habit.total_count += 1

        # 计算连续天数
        today = datetime.now().date()
        if habit.last_date == today - timedelta(days=1):
            # 连续 → streak增加
            habit.streak += 1
        elif habit.last_date != today:
            # 断开 → streak重置
            habit.streak = 1

        habit.last_date = today

        # 计算习惯强度
        habit.strength = self._compute_strength(habit)

        self._habits[habit_name] = habit

    def _compute_strength(self, habit: Habit) -> float:
        """计算习惯强度。"""
        # Streak权重（60%）
        streak_score = min(habit.streak / 14.0, 1.0)  # 14天=满分

        # 总次数权重（40%）
        count_score = min(habit.total_count / 50.0, 1.0)  # 50次=满分

        return 0.6 * streak_score + 0.4 * count_score

# 习惯数据结构：
class Habit:
    name: str
    streak: int           # 连续天数
    total_count: int      # 总次数
    last_date: date       # 最后执行日期
    strength: float       # 习惯强度 ∈ [0, 1]
```
