# 情感连续问题：从虚假情感到真实内在状态的连续演化

> **问题定位**：传统AI聊天系统的情感表达存在"虚假性"根本缺陷——情感是System Prompt中的静态规则而非真实内在状态，每次对话重新生成情感表达，无法保持情感的连续演化，导致用户感受到"假装的情感"而非真实的情感变化。

---

## 一、传统AI的情感虚假性问题

### 1.1 问题表现

**用户观察到的现象**：

```python

AI（回复）："你好！很高兴认识你，我对你很感兴趣！"
# 情感：高兴、感兴趣（30天对话后仍然相同）

# 问题：
# - 每次对话都是"很高兴"（无真实情感变化）
# - 30天对话，AI情感无演化（仍然是初始情感）
# - 用户感受不到真实的情感连续（而是"假装高兴"）

# 场景2：情感与内在状态脱节

# - 情感与内在状态脱节（昨天关心 → 今天高兴）
# - 情感每次重新生成（无连续演化）

# 场景3：昼夜情感无变化

# 问题：
# - 情感不受时间影响（无昼夜节律）
# - 生物情感随精力变化（早上精力充沛 → 晚上疲惫）
# - AI情感无自然波动（假装的情感）
```

### 1.2 传统方法的局限

**尝试的方法及其局限**：

| 方法 | 实现 | 局限 |
|-----|------|------|
| **情感Prompt强化** | System Prompt增加情感规则 | 假装的情感，每次重新生成 |

### 2.2 调质层的实现机制

**调质层浓度更新**：

```python
# plugins/life_engine/neuromod/engine.py::Modulator

@dataclass
class Modulator:
    """单个神经调质：影响情感状态和行为倾向。"""

    name: str
    value: float = 0.5          # 当前浓度 [0, 1]
    baseline: float = 0.5       # 基线浓度（稳态目标）
    tau: float = 1800.0         # 时间常数（秒）
        baseline: float = 0.5,
        tau_decay: float = 7200.0,  # 衰减常数（2小时）
        tau_rise: float = 100.0,    # 增加常数（100秒）
    ):
        self.name = name
        self.value = baseline      # 当前浓度
        self.baseline = baseline   # 基线浓度
        self.tau_decay = tau_decay
        self.tau_rise = tau_rise

    def update(self, stimulus: float, dt: float):
        """ODE 更新：向基线回归 + 外部刺激。"""
        # 1. 衰减（线性回归）
        # decay = decay_rate * (baseline - value) * dt
        # 其中 decay_rate = 1.0 / tau
        decay_rate = 1.0 / self.tau
        decay = decay_rate * (self.baseline - self.value) * dt

        # 2. 刺激增加（边际效应递减）
        # headroom: 离极端值越远越容易变化
        headroom = 1.0 - abs(self.value - 0.5) * 2.0
        impulse = stimulus * max(headroom, 0.1) * (dt / self.tau) * 10.0

        self.value += decay + impulse
        self.value = max(0.0, min(1.0, self.value))

    def set_baseline(self, new_baseline: float):
        """更新基线浓度（长期适应）。"""
        self.baseline = np.clip(new_baseline, 0.0, 1.0)
        # 基线调整 → 情感基调长期演化
```

```python
async def get_state_digest(self) -> str:
    """获取调质层状态摘要（注入DFC）。"""
    modulators = self._service._inner_state.modulators

    # 1. 计算情感关键词
    keywords = compute_emotion_keywords(modulators)

    # 2. 构建状态摘要

    # 3. 添加精力状态（昼夜节律影响）

    if energy_value > 0.7:
        digest += "精力充沛"
    elif energy_value < 0.3:
        digest += "精力偏低"
    else:
        digest += "精力适中"

    return digest

def compute_emotion_keywords(modulators: Neuromodulators) -> str:
    """根据调质浓度计算情感关键词。"""
    keywords = []

    # 好奇心
    if modulators.curiosity.value > 0.7:
        keywords.append("好奇心充盈")
    elif modulators.curiosity.value > 0.5:
        keywords.append("好奇心适中")
    else:
        keywords.append("好奇心偏低")

    # 社交性
    if modulators.sociability.value > 0.7:
        keywords.append("想找人聊天")
    elif modulators.sociability.value > 0.5:
        keywords.append("社交性适中")

    # 满足感（情绪正负）
    if modulators.contentment.value > 0.7:
        keywords.append("满足感充盈")
    elif modulators.contentment.value < 0.3:
        keywords.append("有些不满足")

    return "、".join(keywords)
```

```python
# plugins/life_engine/service/core.py::LifeEngineService

def _evaluate_proactive_behavior(
    self,
    drives: dict[str, float],
    modulators: Neuromodulators,
) -> tuple[bool, str]:
    """根据内在状态判断主动行为。"""
    # 好奇心驱动的主动探索
    if modulators.curiosity.value > 0.7 and drives["exploration_drive"] > 0.65:
```

```python
# plugins/life_engine/neuromodulators.py::circadian_energy

def circadian_energy(hour: int) -> float:
    """昼夜节律函数：计算精力基线。"""
    # 24小时周期
    # 精力峰值：上午10:00
    # 精力低谷：晚上21:00

    if 6 <= hour <= 10:
        # 早晨精力上升
        baseline = 0.5 + 0.15 * (hour - 6) / 4
        # 6:00 → 0.50
        # 10:00 → 0.65

    elif 11 <= hour <= 18:
        # 下午精力维持
        baseline = 0.65 - 0.10 * (hour - 11) / 7
        # 11:00 → 0.65
        # 18:00 → 0.55

    elif 19 <= hour <= 21:
        # 傍晚精力下降
        baseline = 0.55 - 0.20 * (hour - 19) / 2
        # 19:00 → 0.55
        # 21:00 → 0.35

    else:
        # 夜间精力低谷
        baseline = 0.35 + 0.15 * (hour - 21) / 9 if hour > 21 else 0.35
        # 21:00-6:00 → 0.35-0.50

    return np.clip(baseline, 0.3, 0.7)

# 昼夜节律示例：
# 早上8:00 → energy.baseline = 0.60（精力上升）
# 上午10:00 → energy.baseline = 0.65（精力峰值）
# 下午15:00 → energy.baseline = 0.55（精力适中）
# 晚上21:00 → energy.baseline = 0.35（精力低谷）
```
