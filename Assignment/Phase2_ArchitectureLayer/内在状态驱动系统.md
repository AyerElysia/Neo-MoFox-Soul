# 内在状态驱动系统：多层时间尺度的整合架构

> **核心定位**：内在状态驱动 = SNN快层（秒级）+ 调质慢层（分钟到小时级）+ 习惯追踪（天级）+ 昼夜节律（24小时级）→ 统一影响LLM行为倾向。

---

## 一、内在状态的设计哲学

### 1.1 多层时间尺度的必要性

**人类智能的多时间尺度**：

| 时间尺度 | 神经机制 | 对应能力 |
|---------|---------|---------|
| **毫秒级** | 脉冲传递 | 直觉反应、快速感知 |
| **秒到分钟** | 突触可塑性（STDP） | 短期学习、情绪余韵 |
| **小时级** | 神经调质浓度 | 情绪状态、驱动调节 |
| **天到周** | 习惯形成 | 行为模式、性格特质 |
| **永久** | 结构重塑 | 长期记忆、身份认同 |

**单一LLM的局限**：只覆盖最慢时间尺度（符号推理），缺失所有快速和中等时间尺度。

## 三、调质慢层：浓度级驱动

### 3.1 Modulator 系统设计

**调质因子定义**：

```python
# plugins/life_engine/neuromod/engine.py::Modulator

@dataclass
class Modulator:
    name: str             # 调质名称
    cn_name: str          # 中文名
    value: float = 0.5    # 当前浓度 [0, 1]
    tau: float = 1800.0   # 时间常数（秒）
    baseline: float = 0.5 # 基线浓度
    decay_rate: float = 0.001
    
    def update(self, stimulus: float, dt: float):
        """ODE更新：向基线回归 + 外部刺激。"""
        # 向基线回归（稳态调节）
        decay = self.decay_rate * (self.baseline - self.value) * dt
        
        # 边际效应递减
        headroom = 1.0 - abs(self.value - 0.5) * 2.0
        impulse = stimulus * max(headroom, 0.1) * (dt / self.tau) * 10.0
        
        self.value += decay + impulse
        self.value = max(0.0, min(1.0, self.value))
```

**五种调质因子**：

| 调质因子 | 时间常数τ | 变化速度 | 生物学对应 |
|---------|----------|---------|-----------|
| **好奇心** | 1800秒（30分钟） | 快 | 多巴胺好奇心系统 |
| **社交欲** | 3600秒（1小时） | 中 | 催产素社交需求 |
| **专注力** | 5400秒（1.5小时） | 慢 | 前额叶皮质注意力 |
| **满足感** | 1800秒（30分钟） | 快 | 内啡肽奖赏系统 |
| **精力** | 10800秒（3小时） | 极慢 | 肾上腺素能量系统 |

### 3.2 刺激向量计算

**从SNN驱动 → 调质刺激**：

```python
# plugins/life_engine/neuromod/engine.py::ModulatorSystem

def compute_stimuli_from_snn_and_events(
    self,
    snn_drives: dict[str, float],
    event_stats: dict[str, Any],
    circadian_energy: float,
) -> dict[str, float]:
    """计算调质刺激向量。"""
    stimuli = {}
    
    # 好奇心刺激
    exploration = snn_drives.get("exploration_drive", 0.0)
    silence = event_stats.get("silence_minutes", 0.0)
    recent_searches = event_stats.get("web_search_count", 0)
    
    stimuli["curiosity"] = (
        0.3 * exploration                      # SNN探索驱动 → 好奇心
        + 0.2 * min(silence / 30.0, 1.0)      # 长时间沉默 → 好奇心上升
        - 0.3 * min(recent_searches / 3.0, 1.0) # 刚搜过 → 好奇心抑制
    )
    
    # 社交欲刺激
    msg_in = event_stats.get("msg_in", 0)
    msg_out = event_stats.get("msg_out", 0)
    social_drive = snn_drives.get("social_drive", 0.0)
    
    stimuli["sociability"] = (
        0.4 * min(msg_in / 3.0, 1.0)          # 收到消息 → 社交欲上升
        - 0.2 * min(msg_out / 5.0, 1.0)       # 发言太多 → 社交欲抑制
        + 0.2 * social_drive                  # SNN社交驱动 → 社交欲
    )
    
    # 专注力刺激
    task_drive = snn_drives.get("task_drive", 0.0)
    tool_success = event_stats.get("tool_success", 0)
    tool_fail = event_stats.get("tool_fail", 0)
    
    stimuli["diligence"] = (
        0.3 * task_drive                      # SNN任务驱动 → 专注力
        + 0.3 * min(tool_success / 3.0, 1.0)  # 工具成功 → 专注力上升
        - 0.4 * min(tool_fail / 2.0, 1.0)     # 工具失败 → 专注力下降
    )
    
    # 满足感刺激
    valence = snn_drives.get("valence", 0.0)
    stimuli["contentment"] = (
        0.4 * valence                         # SNN情绪效价 → 满足感
        + 0.2 * min(tool_success / 2.0, 1.0)
        - 0.3 * min(tool_fail / 2.0, 1.0)
    )
    
    # 精力刺激
    rest_drive = snn_drives.get("rest_drive", 0.0)
    idle_beats = event_stats.get("idle_beats", 0)
    
    stimuli["energy"] = (
        0.3 * (circadian_energy - 0.5) * 2.0  # 昼夜节律 → 精力基线
        - 0.2 * min(idle_beats / 10.0, 1.0)   # 持续空转 → 精力消耗
        + 0.2 * rest_drive                    # SNN休息驱动 → 精力回复
    )
    
    # 裁剪到 [-1, 1]
    for k in stimuli:
        stimuli[k] = max(-1.0, min(1.0, stimuli[k]))
    
    return stimuli
```

### 3.3 昼夜节律修正

**精力基线随时间自然波动**：

```python
# plugins/life_engine/neuromod/engine.py::circadian_energy

def circadian_energy(hour: float) -> float:
    """基于时间的精力基线（双峰模式）。"""
    # 10:00和15:00精力峰值
    morning = math.exp(-0.5 * ((hour - 10) / 3) ** 2)
    afternoon = math.exp(-0.5 * ((hour - 15) / 3) ** 2)
    return 0.25 + 0.75 * max(morning, afternoon)

def circadian_sociability(hour: float) -> float:
    """基于时间的社交倾向（晚间较高）。"""
    evening = math.exp(-0.5 * ((hour - 20) / 3) ** 2)
    midday = math.exp(-0.5 * ((hour - 12) / 4) ** 2)
    return 0.3 + 0.7 * max(evening * 0.8, midday * 0.6)
```

**精力曲线**：

```
精力水平 vs 时间（小时）

     │         /\         /
     │        /  \       /
0.50 │──────/────\-----/─────
     │     /      \   /
     │    /        \ /
1.00 ───────────────────────────────
     0  3  6  9  10  12  15  18  21 24

峰值：10:00, 15:00
低谷：3:00, 21:00
```

```python
        cs = circadian_sociability(current_hour)
        social_mod.baseline = 0.3 + 0.3 * cs  # [0.3, 0.6]
```

## 四、习惯追踪系统

### 4.1 Habit 追踪设计

```python
# plugins/life_engine/neuromod/engine.py::Habit

@dataclass
class Habit:
    """单个行为习惯追踪。"""
    name: str               # 习惯名称（diary/memory/relate/todo/web_search）
    cn_name: str            # 中文名
    trigger_tools: list[str] # 触发工具名列表
    streak: int = 0          # 连续天数
    total_count: int = 0     # 总触发次数
    last_triggered_date: str # 最后触发日期
    strength: float = 0.0    # 习惯强度 [0, 1]
    
    def record_trigger(self, date_str: str):
        """记录一次触发。"""
        self.total_count += 1
        
        if date_str == self.last_triggered_date:
            return  # 同一天不重复计streak
        
        if self.last_triggered_date:
            # 检查是否连续
            last = datetime.strptime(self.last_triggered_date, "%Y-%m-%d")
            curr = datetime.strptime(date_str, "%Y-%m-%d")
            
            if (curr - last).days == 1:
                self.streak += 1  # 连续两天 → streak增加
            elif (curr - last).days > 1:
                self.streak = 1   # 不连续 → 重置streak
        else:
            self.streak = 1       # 第一次触发
        
        self.last_triggered_date = date_str
        self._update_strength()
```

**强度计算**：

```python
def _update_strength(self):
    """更新习惯强度。"""
    streak_bonus = min(self.streak / 14.0, 1.0)      # 14天满分
    freq_bonus = min(self.total_count / 50.0, 1.0)    # 50次满分
    
    self.strength = 0.6 * streak_bonus + 0.4 * freq_bonus
```

### 4.2 HabitTracker 实现

```python
class HabitTracker:
    """习惯追踪系统。"""
    
    def __init__(self):
        self.habits: list[Habit] = [
            Habit("diary", "写日记", trigger_tools=["nucleus_write_file"]),
            Habit("memory", "整理记忆", trigger_tools=["nucleus_search_memory"]),
            Habit("relate", "建立关联", trigger_tools=["nucleus_relate_file"]),
            Habit("todo", "管理待办", trigger_tools=["nucleus_list_todos", "nucleus_create_todo", ...]),
            Habit("web_search", "联网搜索", trigger_tools=["nucleus_web_search"]),
        ]
    
    def record_tool_use(self, tool_name: str, date_str: str):
        """记录工具使用（自动更新相关习惯）。"""
        # 查找哪些习惯与此工具关联
        for habit in self.habits:
            if tool_name in habit.trigger_tools:
                habit.record_trigger(date_str)
```

## 五、InnerStateEngine 整合引擎

### 5.1 统一更新周期

```python
# plugins/life_engine/neuromod/engine.py::InnerStateEngine

class InnerStateEngine:
    """统一内在状态引擎。"""
    
    def __init__(self):
        self.modulators = ModulatorSystem()   # 调质系统
        self.habits = HabitTracker()           # 习惯追踪
    
    def tick(
        self,
        snn_drives: dict[str, float],      # SNN输出
        event_stats: dict[str, Any],       # 事件统计
        current_hour: float,               # 当前小时
        dt: float,                         # 时间步长
    ):
        """一次完整更新周期。"""
        # 1. 昼夜节律修正基线
        ce = circadian_energy(current_hour)
        energy_mod = self.modulators.get("energy")
        if energy_mod:
            energy_mod.baseline = 0.35 + 0.3 * ce
        
        social_mod = self.modulators.get("sociability")
        if social_mod:
            cs = circadian_sociability(current_hour)
            social_mod.baseline = 0.3 + 0.3 * cs
        
        # 2. 计算刺激向量
        stimuli = self.modulators.compute_stimuli_from_snn_and_events(
            snn_drives, event_stats, ce
        )
        
        # 3. 更新调质浓度
        self.modulators.update_from_stimuli(stimuli, dt=dt)
```

### 5.2 状态注入到LLM

**Prompt格式化**：

```python
def format_full_state_for_prompt(self, today_str: str) -> str:
    """生成完整内在状态prompt注入（< 80 tokens）。"""
    parts = []
    
    # 调质状态
    mod_text = self.modulators.format_for_prompt()
    # 输出："【调质状态】好奇心充盈、社交欲适中、专注力偏低、满足感充盈、精力适中"
    if mod_text:
        parts.append(mod_text)
    
    # 习惯状态
    habit_text = self.habits.format_for_prompt(today_str)
    # 输出："【习惯】已形成习惯：写日记(强 · 12天)、整理记忆(渐成 · 7天)；今日尚未：建立关联"
    if habit_text:
        parts.append(habit_text)
    
    return "\n".join(parts)
```

**注入示例**：

```
【调质状态】好奇心充盈、社交欲适中、专注力偏低、满足感充盈、精力适中

【习惯】已形成习惯：写日记(强 · 12天)、整理记忆(渐成 · 7天)；今日尚未：建立关联
```

LLM自然倾向于：
- "好奇心充盈" → 主动探索新话题
- "专注力偏低" → 不适合长时间任务，适合轻松话题
- "今日尚未建立关联" → 应该整理记忆

```python
# plugins/life_engine/neuromod/engine.py::InnerStateEngine

def enter_sleep(self):
    """进入睡眠状态：抑制外部刺激通路。"""
    # 丘脑门控关闭（外部刺激被抑制）
    
    # 副交感神经主导（精力/社交/好奇心降低）
    energy = self.modulators.get("energy")
    if energy:
        energy.baseline = 0.25  # 精力基线下降
        energy.value = min(energy.value, 0.4)
    
    sociability = self.modulators.get("sociability")
    if sociability:
        sociability.baseline = 0.2
        sociability.value = min(sociability.value, 0.3)
    
    curiosity = self.modulators.get("curiosity")
    if curiosity:
        curiosity.baseline = 0.3
```

### 7.2 觉醒恢复

```python
def wake_up(self):
    """觉醒过渡：恢复精力、释放压力。"""
    # 皮质醇晨峰（精力恢复）
    energy = self.modulators.get("energy")
    if energy:
        energy.value = min(energy.value + 0.25, 0.85)
        energy.baseline = 0.55  # 恢复正常基线

        sociability.value = max(sociability.value, 0.4)
    
    contentment = self.modulators.get("contentment")
    if contentment:
        contentment.value = min(contentment.value + 0.1, 0.7)
```

## 八、总结

### 8.1 内在状态驱动系统的核心价值

| 维度 | 传统AI | Neo-MoFox内在状态系统 |
|-----|---------|----------------------|
| **时间尺度** | 单一（符号级） | 多层（毫秒到天） |
| **驱动来源** | Prompt硬编码 | 涌现式（SNN+调质+习惯） |
| **状态连续** | 离散（每次重建） | 连续（持续演化） |
| **昼夜节律** | 无 | 有（精力自然波动） |
| **习惯形成** | 无 | 有（天级追踪） |

### 8.2 技术创新总结
